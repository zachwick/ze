Overview

ze is a small, extensible text editor that embeds GNU Guile for scripting. It supports a lightweight core, ANSI terminal rendering, syntax highlighting, template cloning, a forward-search UI, and a simple Scheme-based hooks and plugin system for customization.

For the project homepage and published API docs, see the repository README and project site.

Features

  • Simple, minimal core written in C
  • Extensible via GNU Guile Scheme (plugins and hooks)
  • Syntax highlighting for C, Python, Ruby, PHP, Rust, APL, Swift, and TypeScript
  • Template system for quickly bootstrapping files
  • Forward search with next/previous navigation
  • Cross-platform on Unix-like systems (macOS, Linux, …)

Build and Install

Prerequisites

  • GNU Guile installed on your system (e.g., on macOS: brew install guile).

Steps

git clone git@github.com:zachwick/ze
cd ze
mkdir -p ~/.ze
cp zerc.scm ~/.ze/
cp -R templates ~/.ze/templates
make install

You must adjust the path passed to scm_c_primitive_load in main to point to your local zerc.scm if it differs from the default.

Usage

  • Start: ze
  • Open a file: ze <filename>
  • Press Ctrl+x to open a one-line Scheme REPL inside ze
  • Common keys: Ctrl+o (open), Ctrl+w (write), Ctrl+t (clone template), Ctrl+s (search), Ctrl+n/Ctrl+b (next/prev match)

Plugins and Hooks

Place .scm files under ~/.ze/plugins. Non-hidden Scheme files are loaded at startup. You can bind keys and set the status line from Scheme. Hooks are invoked around editor events such as saving and opening files or directories.
